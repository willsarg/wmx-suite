Kokoro TTS Benchmarks

Real-time speech synthesis throughput and factor benchmarking on Apple Silicon.

⏱️ Real-Time Factor (RTF)

Calculated as Compute Time / Audio Output Duration. A value of 1.0 means real-time speed. A value of 0.04 means the audio generates 25x faster than real-time. Lower is better.

⚡ Characters Per Second (CPS)

Calculated as Utterance Length (char) / Compute Time. Measures the character processing throughput. Allows easy scaling comparison across different sentence sizes. Higher is better.
