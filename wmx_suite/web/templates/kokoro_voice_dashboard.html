🔊 Same-Voice Baseline

Sequential synthesis of the same voice style already loaded in memory. Represents the warm execution speed limit. Lower is better.

🔄 Warm Voice Switch

Alternating between different voice style vectors that are already cached in RAM/Metal. Measures internal dictionary lookup and slicing latency. Lower is better.

💾 Cold Voice Load

Clearing the cache and reading a voice style vector from disk. Measures file system parsing and I/O loading latency. Lower is better.
