📈 Batch Concurrency

By running synthesis requests in parallel across multiple threads, we bypass the python lock and submit commands concurrently to the Metal GPU. This improves GPU utilization and aggregate throughput.

⚡ Aggregate Characters Per Second (CPS)

Calculated as Total Characters Generated / Total Elapsed Wall Time. This measures the true aggregate processing power of the concurrent system. Higher is better.
